## HTML

1. HTML의 목적: **브라우저에게 구조를 알려준다**.

2. tag를 이용하여 요소를 삽입한다.

3. HTML 특징

- 잘못된 태그: **오류가 발생하지 않는다.**

- 태그 안 attribute: 태그에 추가적인 정보를 주는 요소. 마찬가지로 **잘못된 attribute를 사용해도 오류가 발생하지 않는다**.
- Self-Closing Tag: 닫는 태그가 없는 태그. **attribute가 태그의 내용이 되므로** 열고 닫는 태그 사이에 text를 넣을 필요가 없음.
